Description

Ce projet a été réalisé pour l'entreprise Planification MASA, une entreprise de gestion funéraire qui accompagne ses clients pour les guider dans un processus complexe à la suite d'un décès.

L'objectif était de refléter leur image à travers un site web en intégrant non seulement leur guide de style, mais aussi leurs caractéristiques distinctives. De plus, j'ai assuré un suivi régulier avec le client pour m'assurer que tout correspondait à leurs besoins.

Logiciels et langages

  • Figma
  • Visual Studio Code
  • Photoshop

Processus

Pour commencer, j’ai fait plusieurs essais pour le type de design. Une fois le style trouvé, j’ai ajouté des images pour représenter chaque section.

Ensuite, je suis passée à l’intégration où j’ai réalisé le responsive. Puis, j’ai complété par quelques animations pour le menu et les liens.
