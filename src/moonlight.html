Moonlight

Projet scolaire

Design d'une application de type revue

Prototypage

Description

Moonlight est un projet de design réalisé dans le cadre d'un cours. Il s'agit d'une application permettant de partager des articles, à la manière d'une revue en ligne, avec pour thème l'astrologie.

J'ai dû créer une édition comprenant plusieurs articles. L'application devait inclure plusieurs pages, notamment celles de démarrage, de connexion, de paramètres, ainsi qu'une page "À propos".

Logiciels et langages

  • Figma

Processus

Tout d'abord, j'ai effectué des recherches sur le thème de l'astrologie pour le représenter de manière adéquate. Ensuite, j'ai conçu les wireframes de toutes les pages, puis j'ai appliqué les styles et ajouté le contenu. Enfin, j'ai réalisé le prototypage du design.
